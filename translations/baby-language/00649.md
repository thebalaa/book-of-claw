## The Translation

SSH has a default key location (`~/.ssh/id_rsa` or similar). If you don't specify a custom identity, it should use the default.

This fix ensures that when the identity field is blank, SSH falls back to the default key. Plus, the CHANGELOG gets 31 lines documenting recent changes!
