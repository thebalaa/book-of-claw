## The Translation

A big new feature: Telegram support! Not just WhatsApp anymore — now Clawdis can work with Telegram using the grammY framework.

New files:
- `bot.ts` (233 lines) — the Telegram bot
- `send.ts` (88 lines) — sending messages
- `monitor.ts` (63 lines) — watching for messages
- `download.ts` (50 lines) — downloading media
- Tests for all of them!
- Documentation at `docs/telegram.md`

Plus Bun bundle docs for Mac. A major platform expansion!
